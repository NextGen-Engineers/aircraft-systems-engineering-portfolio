## Flagship Project

### Commercial Aircraft Fuel Management System

This project simulates a simplified aircraft fuel management system and follows a safety-critical software engineering workflow:

1. Define stakeholder needs
2. Create system-level requirements
3. Translate system requirements into software requirements
4. Maintain requirements traceability
5. Design system architecture
6. Implement embedded C software components
7. Verify requirements through test cases
8. Analyze safety and reliability risks
9. Document the complete engineering lifecycle
